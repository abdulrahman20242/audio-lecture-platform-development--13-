## ما هي المهارات؟

المهارات حزم معرفية قابلة لإعادة الاستخدام تُوسّع قدرات الـ agent. كل مهارة مجلد يحتوي على ملف `SKILL.md` بتعليمات يتبعها الـ agent تلقائياً.

كل مهارة تحتوي على:

| المكوّن | الوصف |
|---|---|
| **تعليمات المهمة** | إرشادات دقيقة حول كيفية التعامل مع نوع معين من المهام |
| **أفضل الممارسات** | اتفاقيات وأنماط مجرّبة تضمن جودة عمل الـ agent باستمرار |
| **موارد اختيارية** | سكريبتات وقوالب وأمثلة يمكن للـ agent الرجوع إليها |

> **ملاحظة:** لا تحتاج لإخبار الـ agent باستخدام المهارة — يقررها تلقائياً بناءً على سياق محادثتك. يمكنك ذكر اسمها صراحةً إن أردت ضمان تفعيلها.

---

## كيف تعمل المهارات؟

تتّبع المهارات نمط الكشف التدريجي — ثلاث مراحل تحدث تلقائياً في كل محادثة:

### ١ — الاكتشاف
الـ agent يرى قائمة المهارات المتاحة مع أسمائها وأوصافها عند بدء المحادثة.

### ٢ — التفعيل
إذا بدت المهارة ملائمة، يقرأ الـ agent محتوى `SKILL.md` كاملاً.

### ٣ — التنفيذ
يتّبع الـ agent التعليمات بدقة طوال فترة العمل على المهمة.

## خطوات التثبيت

> ⚠️ يستخدم Antigravity `.agents/skills` افتراضياً. لا يزال يدعم `.agent/skills` للتوافق مع الإصدارات القديمة.

### الخطوة ١ — اختر موقع المهارة

حدد أين ستعيش مهارتك:

| المسار | النطاق |
|---|---|
| `<workspace>/.agents/skills/` | خاص بالمشروع |
| `~/.gemini/antigravity/skills/` | عالمي (جميع المشاريع) |

---

### الخطوة ٢ — أنشئ مجلد المهارة

اصنع مجلداً باسم مهارتك داخل مسار المهارات المختار:

```bash
mkdir -p .agents/skills/my-skill
```

---

### الخطوة ٣ — أنشئ ملف SKILL.md

يحتوي على YAML frontmatter ثم التعليمات التفصيلية للـ agent:

```yaml
---
name: my-skill
description: وصف واضح لما تفعله المهارة ومتى تُستخدم.
---

# اسم المهارة

تعليمات تفصيلية للـ agent.

## متى تُستخدم
- استخدمها عند...

## كيفية الاستخدام
خطوات وأنماط يتبعها الـ agent.
```

> 💡 اكتب الـ `description` بصيغة الغائب مع كلمات مفتاحية — هذا ما يقرأه الـ agent ليقرر تفعيل المهارة.

---

### الخطوة ٤ — أضف موارد إضافية (اختياري)

يمكن إثراء المهارة بمجلدات إضافية يستطيع الـ agent قراءتها عند الحاجة:

```
.agents/skills/my-skill/
├── SKILL.md        # مطلوب دائماً
├── scripts/        # سكريبتات مساعدة
├── examples/       # أمثلة مرجعية
└── resources/      # قوالب وأصول
```

---

### الخطوة ٥ — تحقق من التثبيت

ابدأ محادثة جديدة وتأكد أن المهارة تظهر في قائمة المهارات:

```bash
antigravity skills list

✓  my-skill          → .agents/skills/my-skill/
✓  code-review       → .agents/skills/code-review/
✓  deploy-helper     → ~/.gemini/antigravity/skills/deploy-helper/
```

## كيفية الاستخدام

### الاستخدام التلقائي

لا تحتاج لأي إجراء — الـ agent يكتشف المهارة الملائمة ويفعّلها من تلقاء نفسه:

```
أنت  ← راجع هذا الـ PR وأخبرني إذا كان هناك أخطاء

Agent ← ✓ يكتشف مهارة "code-review" ويفعّلها تلقائياً
         سأراجع الكود بناءً على قائمة التحقق...
```

### الاستخدام الصريح

ذكر اسم المهارة مباشرةً يضمن تفعيلها:

```
أنت  ← استخدم مهارة code-review لفحص هذا الملف

Agent ← ✓ يقرأ SKILL.md ويطبق التعليمات مباشرةً
         سأتبع قائمة التحقق المحددة في المهارة...
```

### 02 — React Best Practices — Vercel Labs

**النوع:** Skill / خبرة

خبرة مدرّبة على أفضل ممارسات React من Vercel، تُضاف للنموذج لتوجيهه نحو كتابة كود نظيف وقابل للتوسع — تشمل قواعد Components وHooks وبنية المشروع.

```bash
# البحث عن خبرة بالاسم
npx skills find

# إضافة الخبرة من Vercel Labs
npx skills add vercel-labs/agent-skills
```

🔗 [قراءة المزيد على Vercel Blog](https://vercel.com/blog/introducing-react-best-practices)

### 03 — Frontend Design — Anthropic Skills
