# أوامر التشغيل والبناء — منصة المحاضرات الصوتية

هذا المستند يحتوي على الأوامر الأساسية لتشغيل وتطوير وبناء المشروع.

## 🚀 بيئة التطوير (Development)
لتشغيل المشروع محلياً مع التحديث التلقائي (HMR):
```bash
npm run dev
```

## 🏗️ البناء للإنتاج (Production Build)
لإنشاء نسخة جاهزة للنشر في مجلد `dist`:
```bash
npm run build
```

## 🔍 معاينة البناء (Preview)
لمعاينة النسخة المبنية (`dist`) محلياً قبل النشر:
```bash
npm run preview
```

---
💡 **ملاحظة**: تأكد من تثبيت الحزم باستخدام `npm install` قبل التشغيل لأول مرة.
