التعريف
ما هي المهارات؟

المهارات حزم معرفية قابلة لإعادة الاستخدام تُوسّع قدرات الـ agent. كل مهارة مجلد يحتوي على ملف SKILL.md بتعليمات يتبعها الـ agent تلقائياً.

تعليمات المهمة

إرشادات دقيقة حول كيفية التعامل مع نوع معين من المهام.

أفضل الممارسات

اتفاقيات وأنماط مجرّبة تضمن جودة عمل الـ agent باستمرار.

موارد اختيارية

سكريبتات وقوالب وأمثلة يمكن للـ agent الرجوع إليها.

لا تحتاج لإخبار الـ agent باستخدام المهارة — يقررها تلقائياً بناءً على سياق محادثتك. يمكنك ذكر اسمها صراحةً إن أردت ضمان تفعيلها.

آلية العمل
كيف تعمل المهارات؟

تتّبع المهارات نمط الكشف التدريجي — ثلاث مراحل تحدث تلقائياً في كل محادثة.

١
الاكتشاف

الـ agent يرى قائمة المهارات المتاحة مع أسمائها وأوصافها عند بدء المحادثة.

٢
التفعيل

إذا بدت المهارة ملائمة، يقرأ الـ agent محتوى SKILL.md كاملاً.

٣
التنفيذ

يتّبع الـ agent التعليمات بدقة طوال فترة العمل على المهمة.

التثبيت
خطوات تثبيت المهارات

خمس خطوات فقط تفصلك عن تشغيل أولى مهاراتك في Antigravity.

يستخدم Antigravity .agents/skills افتراضياً. لا يزال يدعم .agent/skills للتوافق مع الإصدارات القديمة.

١
اختر موقع المهارة

حدد أين ستعيش مهارتك — خاص بالمشروع أو عالمي لجميع مشاريعك.

المسارالنطاق
<workspace>/.agents/skills/ خاص بالمشروع
~/.gemini/antigravity/skills/ عالمي
٢
أنشئ مجلد المهارة

اصنع مجلداً باسم مهارتك داخل مسار المهارات المختار.

Terminal
mkdir -p .agents/skills/my-skill
٣
أنشئ ملف SKILL.md

يحتوي على YAML frontmatter ثم التعليمات التفصيلية للـ agent.

SKILL.md
--- name: my-skill description: وصف واضح لما تفعله المهارة ومتى تُستخدم. --- # اسم المهارة تعليمات تفصيلية للـ agent. ## متى تُستخدم - استخدمها عند... ## كيفية الاستخدام خطوات وأنماط يتبعها الـ agent.

اكتب الـ description بصيغة الغائب مع كلمات مفتاحية — هذا ما يقرأه الـ agent ليقرر تفعيل المهارة.

٤
أضف موارد إضافية (اختياري)

يمكن إثراء المهارة بمجلدات إضافية يستطيع الـ agent قراءتها عند الحاجة.

.agents/skills/my-skill/ ├── SKILL.md # مطلوب دائماً ├── scripts/ # سكريبتات مساعدة ├── examples/ # أمثلة مرجعية └── resources/ # قوالب وأصول
٥
تحقق من التثبيت

ابدأ محادثة جديدة وتأكد أن المهارة تظهر في قائمة المهارات.

Terminal
antigravity skills list ✓ my-skill → .agents/skills/my-skill/ ✓ code-review → .agents/skills/code-review/ ✓ deploy-helper → ~/.gemini/antigravity/skills/deploy-helper/
الاستخدام
كيفية استخدام المهارات

بعد التثبيت، يتعامل الـ agent مع المهارات تلقائياً. إليك أبرز أنماط الاستخدام.

الاستخدام التلقائي

لا تحتاج لأي إجراء — الـ agent يكتشف المهارة الملائمة ويفعّلها من تلقاء نفسه.

محادثة مع الـ Agent
أنت ← راجع هذا الـ PR وأخبرني إذا كان هناك أخطاء Agent ← ✓ يكتشف مهارة "code-review" ويفعّلها تلقائياً سأراجع الكود بناءً على قائمة التحقق...
الاستخدام الصريح

ذكر اسم المهارة مباشرةً يضمن تفعيلها حتى لو لم يخترها الـ agent من تلقاء نفسه.

محادثة مع الـ Agent
أنت ← استخدم مهارة code-review لفحص هذا الملف Agent ← ✓ يقرأ SKILL.md ويطبق التعليمات مباشرةً سأتبع قائمة التحقق المحددة في المهارة...
إضافات جاهزة
٣ مهارات لا غنى عنها في البرمجة بالـ AI

أدوات ومهارات جاهزة ستغيّر طريقة عملك مع الذكاء الاصطناعي في التطوير.
